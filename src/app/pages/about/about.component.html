About User-post-album app

This app provides a way to view the posts and albums a user produces. It has the following functions:

  • Create: Ability to create new posts and albums for a user. To do this, select a user, press load albums or load posts, then select Add and create your new item.
  • Read: Retrieves a list of users, posts and albums.
  • Update: Update users email address via the view user page. To do this, select a user, and press the edit pencil next to the email. Finally press the green update button to save.
  • Delete: Ability to delete users via the view user page. To do this, select a user and press Delete User at the top of the page.
